Mixins

Mixins

Let’s try a mixin now! We can use one for list styling, let’s call it list-reset.

  1. In the project pane on the left, right click on the sass folder and select New File.

  2. Type _mixins.scss and press enter. This will be the partial file where any mixins we us in our project will be located.

  3. Don't forget to import mixins into your styles.scss file. Since we'll be using some of these mixins in the _nav.scss and _footer.scss files it will need to be imported before those files are imported.

  4. Add your mixin to the _mixins.scss file. This is adding a “function” to set the three properties list-style, margin and padding anytime this mixin is used in your Sass file:

     @mixin list-reset() {
     list-style-type: none;
     padding: 0;
     margin: 0;
 }

  5. In your _nav.scss and _footer.scss files, replace the list-style-type: none; code with @include list-reset();

  6. Save your _nav.scss, _footer.scss, styles.scss and _ _mixins.scss. files.

  7. In the command line type sass sass/styles.scss css/styles.css.

  8. Go to Google Chrome and refresh your page. It should look the same but now you're consistently setting list styles across the project and have a single place to go change them should you need to in the future.
