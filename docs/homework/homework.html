Hey Slacker!

Remember, we're here to help. Join the KCWiT #codingandcocktails Slack Channel: kcwit.slack.com

The more you practice, the more comfortable you’ll feel! Take a look at the following tutorials:

  1. Practice more CSS styling on Codecademy: https://www.codecademy.com/courses/css-coding-with-style

  2. Play a game to understand CSS Selectors better: http://flukeout.github.io/

  3. To understand how CSS Grid works try: http://cssgridgarden.com/

  4. To understand how Flexbox works try: http://flexboxfroggy.com/

  5. For a deeper dive into CSS Grid try: https://cssgrid.io/

Check out these tools designed to help web developers

  1. Deploy your webpage to GitHub as a GitHub page. Reach out on Slack and mentor will be happy to walk you through it. We'll learn about Git next session so you're getting a head start!
  2. Using Chrome DevTools, run Lighthouse audit for "Accessibility" to generate a report.
  3. View your page in Google Resizer to visualize what it looks like in different form factors.
